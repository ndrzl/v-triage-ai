# V-Triage AI
**Track 3: Vital Signs (Healthcare & Wellbeing)**

V-Triage AI is an intelligent emergency prioritization system designed to reduce overcrowding in Malaysian ERs by categorizing patient urgency using AI-driven symptom analysis.

### Tech Stack
- **Framework:** Flask (Python)
- **Model:** Google Gemini 1.5 Flash
- **Infrastructure:** Google Cloud Ecosystem
